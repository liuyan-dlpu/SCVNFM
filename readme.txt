The code for each dataset is placed in the folder named after the corresponding dataset.
For cases where some methods are not included within a dataset-specific folder, please refer to the mainV1.m main script located in the Others directory.
The associated subroutines are: SCVNFMV1, SCVNFMV1_NoL2, SCVNFMV1_NoMomentum, and SCVNFMV1_NoMomentumL2.
